This program creates visual representations of times and places. It'll create a unique piece of art for any occasion that is special to you.

First find the GPS coordinates of the place for which you want a visual representation (using a website like this for example) and then plug them into the the corresponding fields. You can also add a time and date. When you're ready, click the 'submit' button and the program will generate visuals that represent the numbers you've input :)

This is the first program I've ever put out there and I'm really happy about it! There is ample room for improvement and I'm ok with that. E-mail me with any bugs, suggestions or whatever else :) lapoiremolle@gmail.com
